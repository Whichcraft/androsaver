# Default Images

Place image files here (JPEG, PNG, WebP, GIF, BMP).

These images are bundled inside the APK and shown automatically in Photo Slideshow mode
when no image source (Google Drive, OneDrive, Dropbox, Immich, Nextcloud, Synology, Device
Photos) is enabled. Once any source is enabled this folder is ignored.

**To add images:** drop files here and commit. They will be included in the next build.

Recommended: a handful of high-quality landscape photos at 1920×1080 or higher.
